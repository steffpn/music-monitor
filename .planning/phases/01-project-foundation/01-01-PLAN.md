Create the `packages/shared` package with all v1 type definitions. These types define the data contract between backend and iOS app for the entire project.

**packages/shared/package.json:**
- `"name": "@myfuckingmusic/shared"`, `"version": "0.0.1"`, `"private": true`, `"type": "module"`
- `"main": "./src/index.ts"`, `"types": "./src/index.ts"`
- Scripts: `build` (tsc), `test` (vitest run)
- devDependencies: typescript ^5.7.0, vitest ^3.0.0

**packages/shared/tsconfig.json:**
- Extends `@myfuckingmusic/tsconfig/base.json`
- compilerOptions: outDir ./dist, rootDir ./src
- include: ["src/**/*"]

**Type files -- create all with full v1 type definitions:**

1. `packages/shared/src/enums/roles.ts`:
   - `UserRole` enum: ADMIN, ARTIST, LABEL, STATION
   - `InvitationStatus` enum: PENDING, REDEEMED, EXPIRED, REVOKED

2. `packages/shared/src/enums/status.ts`:
   - `StreamStatus` enum: ACTIVE, INACTIVE, ERROR, RESTARTING
   - `DetectionStatus` enum: RAW, AGGREGATED

3. `packages/shared/src/types/station.ts`:
   - `Station` interface: id (number), name (string), streamUrl (string), status (StreamStatus), country (string, default "RO"), stationType ("radio" | "tv"), lastHeartbeat (Date | null), restartCount (number), createdAt (Date), updatedAt (Date)
   - `StationCreate` interface: name, streamUrl, stationType, country?
   - `StationUpdate` interface: Partial of name, streamUrl, status, stationType

4. `packages/shared/src/types/detection.ts`:
   - `DetectionEvent` interface: id (number), stationId (number), detectedAt (Date), songTitle (string), artistName (string), albumTitle (string | null), isrc (string | null), confidence (number, 0-100), durationMs (number), rawCallbackId (string | null), createdAt (Date)
   - `DetectionCreate` interface: stationId, detectedAt, songTitle, artistName, albumTitle?, isrc?, confidence, durationMs, rawCallbackId?
   - `AirplayEvent` interface: id (number), stationId (number), startedAt (Date), endedAt (Date), songTitle (string), artistName (string), isrc (string | null), playCount (number, how many raw detections aggregated), snippetUrl (string | null), createdAt (Date)
   - `AirplayCreate` interface: stationId, startedAt, endedAt, songTitle, artistName, isrc?, playCount

5. `packages/shared/src/types/user.ts`:
   - `User` interface: id (number), email (string), name (string), role (UserRole), scopeId (number | null, references artist/label/station), isActive (boolean), lastLoginAt (Date | null), createdAt (Date), updatedAt (Date)
   - `UserCreate` interface: email, name, role, scopeId?
   - `UserPublic` interface: id, name, role (omit sensitive fields)

6. `packages/shared/src/types/snippet.ts`:
   - `AudioSnippet` interface: id (number), detectionId (number), stationId (number), storageKey (string, R2 object key), durationMs (number, always 5000), encoding (string, "aac"), bitrate (number, 128), sizeBytes (number), presignedUrl (string | null, generated on demand), createdAt (Date)
   - `SnippetCreate` interface: detectionId, stationId, storageKey, sizeBytes

7. `packages/shared/src/types/invitation.ts`:
   - `Invitation` interface: id (number), code (string), role (UserRole), scopeId (number | null), status (InvitationStatus), createdBy (number, admin user id), redeemedBy (number | null), expiresAt (Date), createdAt (Date), redeemedAt (Date | null)
   - `InvitationCreate` interface: role, scopeId?, expiresAt?

8. `packages/shared/src/constants/index.ts`:
   - `SNIPPET_DURATION_MS = 5000`
   - `SNIPPET_ENCODING = "aac" as const`
   - `SNIPPET_BITRATE = 128`
   - `DETECTION_GAP_TOLERANCE_MS = 300000` (5 minutes -- threshold for deduplication)
   - `STREAM_WATCHDOG_INTERVAL_MS = 30000` (30 seconds)
   - `STREAM_RESTART_DELAY_MS = 5000`
   - `JWT_ACCESS_EXPIRY = "15m"`
   - `JWT_REFRESH_EXPIRY = "7d"`
   - `API_VERSION = "v1"`

9. `packages/shared/src/index.ts`:
   - Barrel file re-exporting everything: `export * from "./types/detection.js"`, `export * from "./types/station.js"`, `export * from "./types/user.js"`, `export * from "./types/snippet.js"`, `export * from "./types/invitation.js"`, `export * from "./enums/roles.js"`, `export * from "./enums/status.js"`, `export * from "./constants/index.js"`
   - Use `.js` extensions in all import paths (ESM requirement)
