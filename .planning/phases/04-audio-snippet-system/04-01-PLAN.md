    **segment-resolver.ts:**
    Create `resolveSegments(stationId: number, detectedAt: Date)` that:
    1. Reads `data/streams/{stationId}/` directory
    2. Filters for `.ts` files only
    3. Stats each file for mtimeMs
    4. Sorts by mtime ascending (oldest first)
    5. Each segment covers approximately [mtime - 10000ms, mtime] (10s segments, mtime = write completion time)
    6. Finds segments overlapping [detectedAt - 2500ms, detectedAt + 2500ms]
    7. Returns `{ segments: string[], seekOffsetSeconds: number } | null`
    8. seekOffsetSeconds = max(0, (windowStart - firstSegStart) / 1000)
    9. Returns null if no matching segments found or directory doesn't exist

    Import DATA_DIR from `../services/supervisor/ffmpeg.js`. Use `node:fs/promises` and `node:path`.

    **r2.ts:**
    Create R2 client singleton with upload and presign helpers:
    - `r2Client`: S3Client with `region: 'auto'`, endpoint `https://{R2_ACCOUNT_ID}.r2.cloudflarestorage.com`
    - `uploadToR2(key: string, body: Buffer, contentType?: string)`: PutObjectCommand to R2_BUCKET_NAME
    - `getPresignedUrl(key: string, expiresInSeconds?: number)`: getSignedUrl with GetObjectCommand, default 86400 (24h)
    - Read env vars: R2_ACCOUNT_ID, R2_ACCESS_KEY_ID, R2_SECRET_ACCESS_KEY, R2_BUCKET_NAME
    - Validate env vars are present at import time only when SNIPPETS_ENABLED=true. If SNIPPETS_ENABLED is not 'true', export stub functions that throw clear errors if called (lazy validation).

  <action>
    **snippet.ts:**
    Create BullMQ snippet extraction worker following the same lifecycle pattern as detection.ts and cleanup.ts:

    1. Export `SNIPPET_QUEUE = 'snippet-extraction'` constant
    2. Define job data interface: `{ airplayEventId: number, stationId: number, detectedAt: string }`
    3. `processSnippetJob(data)`:
       a. Check `process.env.SNIPPETS_ENABLED === 'true'`, skip if not
       b. Call `resolveSegments(stationId, new Date(detectedAt))`
       c. If null, log and return (segments unavailable, best-effort)
       d. Create temp file path: `path.join(os.tmpdir(), `snippet-${airplayEventId}-${Date.now()}.aac`)`
       e. Spawn FFmpeg with concat protocol: `concat:${segments.join('|')}`, `-ss ${seekOffsetSeconds}`, `-t 5`, `-vn`, `-c:a aac`, `-b:a 128k`, `-ar 44100`, `-ac 2`, `-f adts`, outputPath
       f. Wait for FFmpeg to complete (wrap in Promise like research pattern)
       g. Read output file as Buffer
       h. Build R2 key: `snippets/${stationId}/${formatDate(detectedAt)}/${airplayEventId}.aac` where formatDate = YYYY-MM-DD
       i. Call `uploadToR2(key, buffer, 'audio/aac')`
       j. Update AirplayEvent: `prisma.airplayEvent.update({ where: { id: airplayEventId }, data: { snippetUrl: key } })`
       k. Log success with structured fields: { airplayEventId, stationId, r2Key }
       l. try/finally: always delete temp file (even on error)
    4. `startSnippetWorker()`:
       - Queue with `createRedisConnection()`
       - Worker with concurrency 2, removeOnComplete: { count: 500 }, removeOnFail: { count: 2000 }
       - Worker 'failed' event logging
       - Return { queue, worker }
