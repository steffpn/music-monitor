<interfaces>
<!-- API endpoints the dashboard consumes (from Plan 02) -->

Auth endpoints (PUBLIC):
```
POST /api/v1/auth/login
  Body: { email: string, password: string }
  Response: { user: { id, email, name, role }, accessToken: string, refreshToken: string }

POST /api/v1/auth/refresh
  Body: { refreshToken: string }
  Response: { accessToken: string, refreshToken: string }

POST /api/v1/auth/logout
  Headers: Authorization: Bearer <accessToken>
  Body: { refreshToken: string }
  Response: { message: "Logged out" }
```

Admin invitation endpoints (ADMIN JWT required):
```
POST /api/v1/admin/invitations
  Headers: Authorization: Bearer <accessToken>
  Body: { role: "ARTIST"|"LABEL"|"STATION"|"ADMIN", scopeId?: number, maxUses?: number }
  Response: { id, code, role, scopeId, maxUses, usedCount, expiresAt, status, createdAt }

GET /api/v1/admin/invitations
  Headers: Authorization: Bearer <accessToken>
  Response: Invitation[]

PATCH /api/v1/admin/invitations/:id/revoke
  Headers: Authorization: Bearer <accessToken>
  Response: { id, status: "REVOKED" }
```

Admin user endpoints (ADMIN JWT required):
```
GET /api/v1/admin/users
  Headers: Authorization: Bearer <accessToken>
  Response: User[] (with scopes array)

PATCH /api/v1/admin/users/:id/deactivate
  Headers: Authorization: Bearer <accessToken>
  Response: { id, isActive: false }

PATCH /api/v1/admin/users/:id/reactivate
  Headers: Authorization: Bearer <accessToken>
  Response: { id, isActive: true }

PATCH /api/v1/admin/users/:id/role
  Headers: Authorization: Bearer <accessToken>
  Body: { role: string }
  Response: { id, role }

PUT /api/v1/admin/users/:id/scopes
  Headers: Authorization: Bearer <accessToken>
  Body: { scopes: Array<{ entityType: string, entityId: number }> }
  Response: { id, scopes }
```
